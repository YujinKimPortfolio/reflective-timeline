# reflective-timeline
A reflection tool for organizing meaningful life experiences - whether it's internships, projects, travel, personal moments - with guided prompts and pattern insights.

## About the project

This project was built as the final deliverable for **SI 491 Independent Study** at the University of Michigan School of Information, Winter 2026, under the supervision of Prof. Steve Oney.

The central design question: *how can interface choices such as scaffolded prompts, timeline structure, surfacing past entries support deeper reflection rather than just information storage?*

The app is designed around three guided reflection questions for each experience:
1. What felt meaningful?
2. What challenged you?
3. What did you learn about yourself?

Entries can be categorized by type (internship, project, travel, personal, etc.), tagged, placed on a year-grouped timeline, and revisited later with appended "later notes" because the richest reflection often happens on the re-reading.

## Running the app

The app is a single HTML file with no build step and no dependencies.
